Sistema de Monitoramento

O "cérebro" digital do projeto roda em um ambiente Python no VS Code, atuando como o elo entre a nuvem e o armazenamento local.

[Print do Console Python]

O aplicativo realiza as seguintes funções:

  • Processamento: Lê os arquivos JSON vindos do broker.
  • Interface: Exibe alertas de "CHEIO", "ACABANDO" ou "VAZIO" no terminal.
  • Registro: Grava automaticamente cada ação (Porta Aberta/Fechada) no Banco de Dados MySQL para consulta futura.
