O Dispositivo (Hardware)

O hardware foi montado utilizando componentes de baixo custo e alta eficiência para garantir o funcionamento 24/7.

[Foto do Hardware Real]

  • Microcontrolador: ESP32 DevKit V1 com Wi-Fi integrado.
  • Atuador: Servo Motor SG90 conectado ao pino D13.
  • Mecanismo: O motor rotaciona 90 graus para liberar a ração e retorna a 0 graus para vedar o reservatório.
