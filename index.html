Home

Sistema inteligente de alimentação pet utilizando ESP32 e monitoramento em tempo real.

Plataforma

Broker: EMQX Cloud (MQTT)

Banco de Dados: MySQL (Local via Python)

Arquitetura: ESP32 → MQTT (EMQX) → Python → MySQL

Dispositivo

Hardware utilizado: ESP32 DevKit V1, Servo Motor SG90 e Protoboard.

O dispositivo controla a abertura da porta (0º a 90º) conforme o nível de ração.

Aplicativo

Script Python no VS Code que processa os dados da nuvem e registra o status "Porta Aberta" ou "Porta Fechada" no banco de dados.
