Bem-vindo ao Monitoramento Pet

Este projeto utiliza tecnologia IoT para garantir que seu pet nunca fique sem comida. Através de um ESP32 e sensores, monitoramos o nível de ração e controlamos a abertura do reservatório em tempo real.

Nossa Plataforma

A inteligência do sistema reside na integração entre hardware e nuvem:

  • Broker: EMQX Cloud utilizando protocolo MQTT.
  • Backend: Script Python com criptografia TLS.
  • Banco de Dados: MySQL para armazenamento de histórico de alimentação.

O Dispositivo (Hardware)

O "cérebro" do alimentador é um ESP32 DevKit V1. Ele controla um Servo Motor SG90:

  • Ângulo 0º: Porta Fechada (Estado de espera).
  • Ângulo 90º: Porta Aberta (Liberando ração).

Monitoramento e Controle

As mensagens recebidas no tópico aula/iot são processadas e exibidas no terminal. O sistema gera alertas visuais de "CHEIO", "ACABANDO" ou "VAZIO" e registra cada movimento da porta no banco de dados.
