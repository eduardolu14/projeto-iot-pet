A Plataforma de Dados

Para que o sistema funcione globalmente, utilizamos uma arquitetura de nuvem moderna baseada no protocolo MQTT.

  • Broker Cloud: Utilizamos o servidor EMQX Cloud para receber e distribuir as mensagens de status do alimentador.
  • Segurança: A conexão é protegida via TLS, garantindo que as informações do sensor e comandos do motor viajem com segurança.
  • Banco de Dados: Todas as informações recebidas são registradas em um banco MySQL, criando um histórico completo de quando o pet foi alimentado.
