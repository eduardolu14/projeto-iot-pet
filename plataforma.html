Arquitetura da Plataforma

A inteligência de dados do nosso projeto é baseada no protocolo MQTT, permitindo uma comunicação leve e ultra-rápida entre o alimentador e o dono.

  • Broker Cloud: EMQX Cloud (Cluster seguro via porta 8883)
  • Protocolo: MQTT com TLS (Criptografia de dados)
  • Armazenamento: Banco de Dados MySQL local para histórico persistente.
